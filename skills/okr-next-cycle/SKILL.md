# OKR Next Cycle

## 执行规则

- 必须基于复盘评分、证据缺口和甲方待确认问题生成建议。
- 可以建议重复 M2 或 M3，也可以进入 M4 或开启下一轮 Boss OKR。
- 下一轮重点必须映射到未完成或低分 KR。
- 对需要甲方确认的问题必须单独列出。
- 不得把下一轮建议当成用户已确认需求。

## 输出格式

输出下一轮 OKR 建议。

必须包含：

- 建议进入的交付幕
- 下一轮重点
- 上级映射
- 需要甲方确认
- 风险提示
